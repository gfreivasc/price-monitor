Como rodar?
-----------
Tenha um banco de dados definido e modifique `settings.py` de acordo

Instale as dependências
```bash
$ pip install -r requirements.txt
```

Depois rode
```bash
$ scrapy crawl monitor
```

Espere e veja os dados populados. That's all, folks.
